# 组件总览

`TUI` 为应用提供了丰富的基础 UI 组件

- 基础组件
    - Layout
    - Color
    - Icon
    - Loading

- 表单组件
    - Form
    - Input
    - Textarea
    - Checkbox
    - Radio
    - Select
    - Switch

- 数据组件
    - Table
    - Pagination
    - Date
    - Tree
    - Progress

- 交互组件
    - Tab
    - Message
    - Popover
    - COnfirm
    - Dialog
    - Backdrop

- 媒体组件
    - Image
    - Video
    - Audio
